# Intelligent Sensing Lab Coursework - React Upgrade

This repository contains a university coursework web project for an Intelligent Sensing Lab website. It now includes a React-based portfolio version at the repository root, while preserving the original `CW1`, `CW2`, and `CW3` coursework iterations so reviewers can see the project evolve over time.

## Live Demo

Open the GitHub Pages demo:

https://zezhengdu7-byte.github.io/Coursework/

The live demo uses the new React interface and reuses the original coursework image assets.

## Project Structure

- `index.html` - React app entry point for GitHub Pages.
- `src/app.js` - React components, state, navigation, application ranking logic, and page content.
- `src/styles.css` - modern responsive styling for the React version.
- `CW1/` - first version of the multi-page static website.
- `CW2/` - second iteration with separated HTML, CSS, and image assets.
- `CW3/` - final iteration with HTML, CSS, JavaScript, images, and print styling.

## React Version

The React version improves the project by adding:

- A single-page application experience with reusable components.
- Modern responsive layout for desktop and mobile screens.
- React state for the internship group ranking workflow.
- Cleaner cards, tables, navigation, and contact sections.
- A clear implementation page explaining the upgrade.

## Original Coursework Version

The original final coursework version is still available in `CW3/`.

Open `CW3/HTML/Homepage.html` in a browser to view the original static site. It includes:

- Home, About, Apply, Visit, Information, Contact, and Design pages.
- Responsive page-specific CSS files.
- JavaScript for the application and information pages.
- Image assets and print stylesheet support.

## Coursework Iterations

`CW1`, `CW2`, and `CW3` represent three rounds of improvement on the same project. They are kept as separate folders to make the development process and changes easy to review.
